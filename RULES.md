# ISLANDS & ROADS ルール解説

このゲーム（オフライン版）の実際の動作にもとづくルールと、公式『カタン（CATAN／Settlers of Catan）基本ゲーム』との違いをまとめたものです。

---

## 1. このゲームのルール（実装どおり）

### プレイ人数
- **固定4人**（あなた＋NPC3人「ミナト・アオイ・ハル」）。人数の増減はできません。

### 盤面
- 19枚の地形タイル：森4・丘3・牧草4・畑4・山3・砂漠1。
- 数字トークンは2〜12（7を除く）。
- 「デフォルト」（毎回同じ固定配置）か「ランダム」（毎回変化）を開始画面で選択。
- 港は9か所：**2:1（木・レンガ・麦・羊・鉱石の各専用）5か所**と**3:1（どの資源でも）4か所**。盤上では桟橋の点線で対応する2頂点が示されます。

### 初期配置
- 配置順は **0→1→2→3→3→2→1→0**（往復／スネークドラフト）。
- 各自が開拓地2・街道2を置きます。**2個目の開拓地**の周囲タイルから初期資源を受け取ります。
- 全員2勝利点からスタート。

### 手番の流れ
1. **ダイスを振る**（必須・最初）。出目の合計の数字タイルに接する建物の所有者が資源を得る（開拓地1枚／都市2枚）。
2. ダイスを振った**後**に、建設・交換・発展カードが可能。
3. ターン終了。

### 建設コストと上限
| 物 | コスト | 上限 | 勝利点 |
|---|---|---|---|
| 街道 | 木1・レンガ1 | 15本 | （最長交易路で+2） |
| 開拓地 | 木1・レンガ1・麦1・羊1 | 5個 | 1点 |
| 都市（開拓地を昇格） | 麦2・鉱石3 | 4個 | 2点 |
| 発展カード | 麦1・羊1・鉱石1 | 山札25枚 | （勝利点カードのみ1点） |

※開拓地は5個まで。都市に昇格させると開拓地の枠が空き、再び建てられます。

### 発展カード（山札25枚の内訳）
- **騎士14・勝利点5・街道建設2・発見2・独占2**。
- **買ったターンは使えず、次の自分の手番から**。**1ターンに1枚だけ**。
- このゲームでは**ダイスを振った後のみ**使用可能（騎士も含む）。
- 騎士＝盗賊を好きな土地へ動かし、接する相手から1枚奪う（置く前に確認、奪う相手も選べる）。
- 街道建設＝無料の街道2本を自分で選んで配置。
- 発見＝銀行から好きな資源2枚。
- 独占＝資源1種を選び全員から集める。
- 勝利点＝伏せたまま1点（本人のみ確認可、勝利時に公開）。
- 騎士を**3枚以上**使うと**最大騎士力＋2点**。

### 盗賊と「7」
- ダイス合計が7：**手札8枚以上**のプレイヤーは半分（切り捨て）を捨てる。
- 振った人が盗賊を**好きな土地**へ移動し、接する相手が複数なら**誰から奪うか選択**（奪うカードはランダム1枚）。
- 盗賊のいる土地は資源を産出しない。

### 交換
- **銀行交換**：通常4:1。港を持つと3:1／2:1。
- **プレイヤー交換**：手番中に「1種類の資源 N枚 → 別の1種類 M枚」を提案。相手（NPC）が諾否を判定し、**OKした相手の中から選んで成立**させる。

### 公開情報（全員に見える）
- 各プレイヤーの手札枚数・開拓地数・都市数・街道数・**未使用の発展カード枚数**（中身は非公開）・使用済み騎士数。
- 最長交易路・最大騎士力の保持者（バッジ表示）。
- 自分の勝利点合計のみ表示、**他プレイヤーの合計勝利点は非公開**（ゲーム終了時に公開）。

### 勝利条件
- 最初に**合計10勝利点**へ到達したプレイヤーの勝ち（隠し勝利点カードを含めて到達した瞬間に勝利）。

### 最長交易路・最大騎士力
- 最長交易路＝連続5本以上で最長の人に+2。同数なら現保持者を優先（先取者有利）。
- 最大騎士力＝騎士3枚以上で最多の人に+2。同数なら現保持者を優先。

## 2. 公式カタン基本ゲームとの違い

| 項目 | このゲーム | 公式カタン | 種類 |
|---|---|---|---|
| 騎士カードを使うタイミング | **ダイスの後のみ** | ダイスを振る**前**にも使える（収穫前に盗賊を動かす戦術が可能） | ⚠️ ルール差 |
| プレイヤー間交換 | 1種類→1種類の単純提案。NPCが自動で諾否。**人間⇔NPCのみ**で交渉の往復なし | 手番プレイヤーが自由に交渉（複数資源の組み合わせ・値引き交渉・何度でも）。プレイヤー同士で活発に取引 | ⚠️ 簡略化 |
| NPC同士の交換 | なし | （対人ゲームなので該当なし。AIなし） | 仕様差 |
| 人数 | 1人プレイ専用・**4人固定**（人間1＋NPC3） | 3〜4人（拡張で5〜6人）の対人 | 仕様差 |
| 勝利点 | 合計10点 | 合計10点 | ✅ 同じ |
| 初期配置順 | 0,1,2,3,3,2,1,0 | 同じ（スネークドラフト） | ✅ 同じ |
| 発展カード構成 | 騎士14・勝利点5・街道建設2・発見2・独占2（25枚） | 同じ | ✅ 同じ |
| 建設コスト | 街道・開拓地・都市・発展カード | 同じ | ✅ 同じ |
| ピース上限 | 道15・開拓地5・都市4 | 同じ | ✅ 同じ |
| 7のルール | 8枚以上で半分廃棄＋盗賊＋1枚奪取 | 同じ | ✅ 同じ |
| 盗賊で奪う相手 | 接する相手から選べる／カードはランダム | 同じ | ✅ 同じ |
| 最長交易路 | 5本以上・同点は現保持者優先 | 同じ | ✅ 同じ |
| 最大騎士力 | 騎士3枚以上・同点は現保持者優先 | 同じ | ✅ 同じ |
| 発展カードの公開情報 | 枚数のみ公開・中身と勝利点は非公開 | 同じ | ✅ 同じ |
| 港 | 2:1×5・3:1×4 | 同じ | ✅ 同じ |
| 発展カードを買ったターンの勝利点カードで勝利 | 可能（合計に即反映） | 可能 | ✅ 同じ |

### 公式にあってこのゲームにない／簡略化されている点
- **自由なプレイヤー間交渉**（複数資源をまとめて交換、相手からの逆提案、値引き交渉）。
- **対人プレイ**（このゲームはNPC戦のオフライン専用。※オンライン版は別途あり）。
- 騎士の**ダイス前プレイ**。

### このゲーム独自の補助・演出
- 盗賊を置く前の**確認ボタン**（置き直し可）。
- 港と対応マスを示す**桟橋の点線**、保有港の一覧表示。
- NPCの**難易度**（やさしい／ふつう／強い）と**速度**（ゆっくり／ふつう／はやい）の選択。
- NPCが直前に建てた建物・街道の**ハイライト表示**。
- 切り替え可能な**複数BGM**。
