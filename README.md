# ISLANDS & ROADS

カタンに着想を得た、ブラウザで遊べるオリジナルの開拓ボードゲームです。

## 起動

Node.js 18以上でローカルサーバーを起動します。

```sh
npm start
```

その後、次のURLを開いてください。

- オフライン版: `http://localhost:4180/index.html`
- オンライン版: `http://localhost:4180/online.html`

`file://` で直接開かず、上記のHTTP URLを使用してください。

## テスト

```sh
npm run test:all
```

Claude Codeで作業する場合は、プロジェクト直下の `CLAUDE.md` に起動方法、
構成、ゲームルール上の必須条件をまとめています。

## 実装済み

- 名前、盤面モード、音を選べるゲーム開始画面
- 固定の初心者向け盤面と、赤数字が隣接しないランダム盤面
- ランダムな19タイルの島
- ダイスと資源産出
- 街道、開拓地、都市の建設
- 銀行との4:1交換
- 3人の自動プレイヤー
- 往復順で行う、開拓地と街道2組ずつの初期配置
- 通話で相談した内容をボタン1つで反映するプレイヤー交換
- 盗賊、7の手札破棄、資源の強奪と産出停止
- 3:1・資源別2:1の港
- 合成BGMとダイス・建設・交換・盗賊の効果音
- 発展カード、最大騎士力、最長交易路
- 開拓地・都市まで建設して勝利を目指す自動プレイヤー
- 道路15本、開拓地5個、都市4個の駒数制限
- 10勝利点での勝利判定
- モバイル表示
